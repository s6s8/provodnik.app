STYLE
Premium travel marketplace UI, dark glass interface, dense card grid, minimal whitespace, large rounded cards, cinematic destination imagery.

SUBJECT
Marketplace page showing live traveler requests where groups are forming to book tours together.

ACTION
User is browsing open group requests and evaluating which group to join.

ENVIRONMENT
Top glass navigation bar identical to homepage.
Below navigation is a glass filter bar with fields: City, Date, Budget, Group Size.
Main content shows a dense grid of request cards.
Each request card contains a destination photo, city name, date, participants joined, target group size, and estimated price per person.
Example overlay text on cards:
“Rome Food Tour — 4 travelers joined — Target group: 6 — Estimated $42 per person.”
Cards have rounded corners and small glass UI overlays with participant avatars.

CAMERA
Straight UI screenshot perspective designed for product presentation slides.

LIGHTING
Dark interface with subtle glow effects and cinematic travel photography lighting.
