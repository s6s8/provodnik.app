STYLE
Dark premium product UI, modern glass panels, large travel photography, clean card layout, Apple-style interface polish.

SUBJECT
Detailed page of a traveler request where a group is assembling to book a tour.

ACTION
User is viewing the request details and considering joining the group.

ENVIRONMENT
Hero banner showing the destination city.
Below the hero is a structured layout with three panels.

Left panel shows request details:
City, date, participants joined, target group size, budget target.

Center section shows participant avatars and a progress bar representing how close the group is to the target size.

Right side panel shows a large rounded “Join Group” button and a card displaying estimated price per person if one more traveler joins.

Below the main section is a list of guide offers.
Each offer card shows guide photo, guide name, rating, and offered price per person.
Buttons allow travelers to accept offer or propose a counter offer.

CAMERA
Wide UI screenshot style suitable for a product design presentation.

LIGHTING
Dark interface with soft glass highlights and cinematic city photography.
