# Stack

## Основной стек

`Next.js + TypeScript + Tailwind CSS + shadcn/ui + Motion + Supabase + TanStack Query + React Hook Form + Zod + Vercel + Playwright + YooKassa + Yandex Maps`

## По слоям

- Frontend: `Next.js`
- Language: `TypeScript`
- UI: `Tailwind CSS + shadcn/ui`
- Animations: `Motion`
- Backend/database/auth/storage: `Supabase`
- Data fetching: `TanStack Query`
- Forms/validation: `React Hook Form + Zod`
- Hosting/deploy: `Vercel`
- Testing: `Playwright`
- Payments in Russia: `YooKassa`
- Maps: `Yandex Maps`
